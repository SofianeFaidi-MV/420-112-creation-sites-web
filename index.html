HTML / CSS — notions essentielles

Le cours couvre la structure d’une page HTML, les liens hypertextes, les polices, les couleurs et images de fond, les formulaires, ainsi que les éléments multimédias audio et video. Les tableaux et la réactivité des pages font aussi partie du parcours.

Séance — Liens

Liens hypertextes

Créer des liens relatifs, absolus, des ancres internes et des liens mailto:.

<nav> <a href="#introduction">Introduction</a> <a href="#services">Services</a> <a href="#contact">Contact</a> </nav>
Séance — Formulaires

Formulaires

Créer des zones de saisie, des listes, des cases à cocher et des boutons d’envoi.

<form> <label for="nom">Nom</label> <input id="nom" type="text" /> <button type="submit">Envoyer</button> </form>
Séance — Tableaux

Tableaux HTML

Structurer un tableau avec des rangées, des cellules d’en-tête et des cellules de données.

<table> <tr> <th>Nom</th> <th>Note</th> </tr> <tr> <td>Sara</td> <td>92</td> </tr> </table>
Séance — Couleurs

Couleurs et arrière-plans

Appliquer des couleurs, améliorer le contraste et utiliser des images de fond de manière lisible.

body { background-color: #0f172a; color: white; background-image: url("fond.jpg"); background-size: cover; }
Séance — Audio / Vidéo

Médias HTML5

Intégrer du son et de la vidéo dans une page Web avec des contrôles de lecture.

<audio controls> <source src="audio.mp3" type="audio/mpeg" /> </audio> <video controls width="420"> <source src="video.mp4" type="video/mp4" /> </video>
Séance — Responsive

Réactivité

Adapter l’affichage d’une page aux petits écrans avec le viewport, Flexbox et les media queries.

<meta name="viewport" content="width=device-width, initial-scale=1.0" /> @media (max-width: 768px) { .menu { flex-direction: column; } }

JS — Conditions

Conditions

La structure conditionnelle if est introduite avec l’idée qu’une accolade est toujours préférable pour rendre le code plus sûr et lisible. fileciteturn1file18L3-L8

JS — Boucles

Boucles

Le cours distingue for, while, do...while, mais aussi for...of et for...in. Il précise aussi que do...while s’exécute au moins une fois. fileciteturn1file3L5-L13 fileciteturn1file5L1-L8

JS — DOM

DOM et manipulation

Le DOM est présenté comme le modèle qui permet d’accéder aux éléments HTML pour modifier textes, styles et attributs, via des méthodes d’accès au document et des propriétés comme innerHTML et textContent. fileciteturn1file15L4-L10 fileciteturn1file7L6-L16 fileciteturn1file0L9-L16

Exercices intégrés dans la page

Voici des exercices affichés directement dans le site, avec le style “notes + exercice + résultat attendu”.

🔗 Exo 1 – Liens internes avec ancres

Crée une page HTML avec trois sections différentes : Introduction, Services et Contact. Ajoute une barre de navigation qui permet d’aller à chaque section.
<nav> <a href="#introduction">Introduction</a> <a href="#services">Services</a> <a href="#contact">Contact</a> </nav> <section id="introduction">...</section> <section id="services">...</section> <section id="contact">...</section>
Résultat attendu : les liens du menu font défiler la page vers la bonne section.

📊 Exo 2 – Tableau de notes

Crée un tableau HTML qui affiche le nom de trois étudiants et leur note finale.
<table> <tr> <th>Étudiant</th> <th>Note</th> </tr> <tr> <td>Adam</td> <td>88</td> </tr> </table>
Résultat attendu : un tableau lisible avec une ligne d’en-tête et des données.

📝 Exo 3 – Formulaire de contact

Crée un formulaire avec un champ nom, un champ courriel et un bouton d’envoi.
<form> <label for="nom">Nom</label> <input id="nom" type="text" /> <label for="courriel">Courriel</label> <input id="courriel" type="email" /> <button type="submit">Envoyer</button> </form>
Résultat attendu : le formulaire affiche deux champs et un bouton.

🧪 Exo 4 – Concaténation de chaînes

Déclare les variables suivantes et affiche une phrase sur une seule ligne avec concaténation.
let nomProduit = "Clavier"; let quantite = 5; let prixUnitaire = 24.99; let message = quantite + " " + nomProduit + " à " + prixUnitaire + "$ l’unité";
Résultat attendu : 5 Clavier à 24.99$ l’unité

🔁 Exo 5 – Répéter un message avec une boucle

Avec une boucle for, affiche cinq fois : J’aime faire de la programmation en JavaScript.
for (let i = 1; i <= 5; i++) { console.log("J’aime faire de la programmation en JavaScript."); }
Résultat attendu : le message apparaît 5 fois.

🧩 Exo 6 – Modifier un titre avec le DOM

Sélectionne un titre avec getElementById() et remplace son contenu.
<h1 id="titre">Ancien titre</h1> <script> const titre = document.getElementById("titre"); titre.textContent = "Bienvenue dans le cours"; </script>
Résultat attendu : le texte du titre change dans la page.
