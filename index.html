Cours 420-112

Création de sites web : HTML, CSS et JavaScript

Cette version de la page est adaptée à partir de ton matériel de cours. On y retrouve une progression qui commence par la structure de base d’un fichier HTML et les propriétés CSS, puis poursuit avec les liens, les tableaux, les formulaires, les couleurs, l’audio/vidéo, la réactivité et un bloc complet sur JavaScript et le DOM.

Voir les documents Aller au bloc JavaScript

Contenu du cours

Le site est organisé en deux grands blocs : les fondements HTML/CSS du développement Web et l’introduction à JavaScript pour rendre les pages interactives.

Bloc 1

HTML et CSS

  • Structure de base d’un fichier HTML
  • Structuration du texte
  • Liens hypertextes et ancres
  • Polices, styles et propriétés CSS
  • Tableaux, formulaires et médias
  • Couleurs, images de fond et réactivité
Bloc 2

JavaScript

  • Introduction à JavaScript
  • Variables, constantes, alert() et prompt()
  • Chaînes de caractères, fonctions mathématiques et dates
  • Conditions, boucles et structures de répétition
  • DOM, fonctions, objets et manipulation de page
