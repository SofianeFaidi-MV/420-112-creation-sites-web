Introduction au Web

Vue d’ensembleFondations

Les trois piliers

  • HTML définit la structure de la page.
  • CSS gère l’apparence, les couleurs, les polices et la mise en page.
  • JavaScript ajoute l’interactivité et la logique.
À retenirUne bonne page Web sépare la structure, la présentation et le comportement.

Exemple concret

Un formulaire de contact est structuré en HTML, stylisé avec CSS et validé avec JavaScript avant l’envoi.

Structure de base d’un fichier HTML

DOCTYPEheadbody

Le squelette minimal

Un document HTML commence par la déclaration du type de document, suivie de la balise racine <html>. La section <head> contient les métadonnées, tandis que <body> contient le contenu visible.

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ma première page</title>
</head>
<body>
  <h1>Bonjour !</h1>
  <p>Bienvenue dans le cours de création de sites Web.</p>
</body>
</html>
ExerciceCrée un fichier index.html contenant un titre principal, un sous-titre et deux paragraphes.

JavaScript et le DOM

getElementByIdinnerHTMLtextContent
<h1 id="titre">Titre initial</h1>
<button onclick="changerTitre()">Changer le titre</button>

<script>
function changerTitre() {
  document.getElementById("titre").textContent = "Titre modifié";
}
</script>
<input id="nom" type="text" placeholder="Votre nom">
<button id="btn">Afficher</button>
<p id="resultat"></p>

<script>
document.getElementById("btn").addEventListener("click", function () {
  const valeur = document.getElementById("nom").value;
  document.getElementById("resultat").innerHTML = "Bonjour <strong>" + valeur + "</strong>";
});
</script>
DifférencetextContent modifie du texte brut, alors que innerHTML peut injecter des balises HTML.
